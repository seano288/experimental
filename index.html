  1. Restart computer and immediately press and hold ⌘S to open a command line.
  2. Once the command has finished loading, type in the following commands, hitting return after each line.

  1. The computer will then reboot. Once finished, insert the required info to set up a new account, but do not import new data or create an Apple ID.
  2. Once finished, it’ll then login to the created account, which will have full admin access.
